# AI Interview Rehearsal

A web app that helps you practice answering common interview questions and get instant feedback on your responses.

## Features

- Choose a role (General, Frontend Developer, Backend Developer) to get relevant interview questions
- Random question generator pulled from a curated question bank
- Instant feedback based on your answer's length and structure
- Clean, responsive UI with smooth animations

## How it works

1. Select a role from the dropdown
2. Click "Get Interview Questions" to receive a random question
3. Type your answer in the text box
4. Click "Submit Answer" to get feedback

## Tech stack

- HTML
- CSS (custom properties, animations, responsive design)
- Vanilla JavaScript (DOM manipulation, event listeners)
